# bulutegitim
 Final Year Project
- Educational Website and Mobile App
# About Project Flow:
 The basic working logic of the project is as follows. Students and lecturers can create accounts. Students can watch and lecturers both watch and upload educational videos. Video uploading procedure is under the control of the admin.
